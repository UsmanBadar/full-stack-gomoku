# Gomoku Game

This program is a web application of Gomoku game including:
1. User authentication with username (admin) and password (admin)
2. Ability to set Board width
3. Gomoku game
4. Previous games saved in database
5. A read only view of previous game board





### Client-Side:

Install dependencies:

### `yarn add` 

Run application: 

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000]to view it in the browser.





### Server-Side:

Install dependencies:

### `yarn add` 

Run application:

### `yarn run dev`

Runs the app in the development mode at [http://localhost:8080].
